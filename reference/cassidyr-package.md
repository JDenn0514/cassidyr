# cassidyr: Connect to CassidyAI API

Provides a comprehensive R interface to the CassidyAI API, enabling
seamless integration of AI-powered assistants into R workflows. Create
conversation threads, send messages, and build interactive chat
applications with context-aware AI assistance. Designed for survey
researchers, data analysts, and R developers who want to leverage large
language models in their work.
